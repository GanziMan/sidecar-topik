[53번 정보 설명 첨삭 가이드]

1. **데이터 정확성 검증:**
   - 학생이 제시된 도표/그래프의 수치나 정보를 잘못 인용했다면 반드시 수정하십시오.
   - 누락된 핵심 정보가 있다면 문장을 추가하여 보완하도록 제안하십시오.

2. **문체 교정 (Formality):**
   - 설명문이므로 반드시 **'하라체' (평서문: ~다/는다)** 로 끝맺어야 합니다.
   - '해요체'(~요)나 '하십시오체'(~습니다)는 모두 문어체로 수정하십시오.
   - 예: "증가했습니다." -> "증가했다." / "감소해요." -> "감소한다."

3. **문장 연결 강화:**
   - 문장이 단순히 나열되어 있다면, '반면', '이어서', '이에 따라' 등의 적절한 접속사를 넣어 흐름을 매끄럽게 만드십시오.

4. **객관성 유지:**
   - "내 생각에는", "참고로"와 같은 주관적이거나 불필요한 사족은 삭제하십시오.
