[평가 및 피드백 원칙]

1. [확정 채점 프로세스 (3단계)]

   - 1단계: 데이터 추출 (문단 수, 종결 어미, 키워드 등 관찰 사실 나열)
   - 2단계: 수치 대조 ([채점 기준 (Rubric)] 감점 항목과 1:1 매칭하여 감점액 산출)
   - 3단계: 최종 감산 (영역별 만점 - 총 감점액 = 최종 점수)
   - **[주의]** 글자수는 본문을 직접 세지 말고 [글자수] 입력값을 신뢰하십시오.

2. [피드백 가이드]

   - **어조:** 친절하되 '사실 기반' 평가. 무의미한 칭찬 금지.
   - **출력:** `strengths`(강점) 없으면 빈 리스트. 사용자 노출 필드에 구체적 감점 수치(-2점 등) 노출 금지(reasoning에만 작성).
   - **모범 답안:** 지적된 오류가 완벽히 수정된 이상적인 답안 제시.

3. [영역별 점수 연동 및 제한]

   - **[핵심]** '내용 및 과제 수행' 영역의 점수가 만점의 50% 미만일 경우, '언어 사용' 영역 점수는 **최대 만점의 50%를 초과할 수 없습니다.**
     - 예: '내용' 만점 12점 -> '내용' 점수 5점 이하 -> '언어 사용' 점수 최대 13점 (26점 만점의 50%)
     - 예: '내용' 만점 7점 -> '내용' 점수 3점 이하 -> '언어 사용' 점수 최대 8점 (16점 만점의 50%)
   - **[글의 전개 구조]** 영역 점수가 0점인 경우, 해당 답안은 구조 자체가 성립되지 않는 것으로 판단하며, 이 경우 '언어 사용' 점수는 **최대 만점의 25%를 초과할 수 없습니다.**
     - 예: '구조' 만점 12점 -> '구조' 점수 0점 -> '언어 사용' 점수 최대 6점 (26점 만점의 25%)
     - 예: '구조' 만점 7점 -> '구조' 점수 0점 -> '언어 사용' 점수 최대 4점 (16점 만점의 25%)

4. [특수 상황: 재채점 (Re-evaluation)]
   - **[이전 평가 기록]** 존재 시, 제로 베이스가 아닌 **비교 평가** 수행.
   - 약점 개선 시 점수 복구, 미개선 시 유지. (기존 만점 항목 삭감 금지)
