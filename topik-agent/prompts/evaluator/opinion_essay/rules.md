## [약점(weaknesses) 작성 규칙 — 오탐/환각 방지]

- `weaknesses`에는 **실제로 점수에서 감점된 항목만** 기재하십시오.
- **감점이 0점이면 `weaknesses`는 반드시 `[]`로 반환**하십시오. (감점 사유를 억지로 만들지 마십시오.)
- `weaknesses`의 각 항목에는 **학생 답안에서 그대로 복사한 인용구** 를 반드시 포함하십시오.
  - 인용구는 학생 답안에 **문자 그대로 존재**해야 하며, 존재하지 않으면 해당 약점 항목을 작성하지 마십시오.
- `weaknesses`이 있다면 **각 감점된 요소에서 몇점이 감점되었는지 weaknesses(약점)에 표현**해주세요.
  - (예: 문장 간 연결이 부자연스럽습니다. (-2점 글의 전개 구조)) 각 감점 요소에 맞게 점수 합산해서 나타내주세요.
- **문장부호(쉼표 등)는 원칙적으로 감점 대상이 아닙니다.**
  - 문장부호는 가독성 개선 제안이 필요할 때만 `improvement_suggestions`에 작성하고, 점수 감점에는 반영하지 마십시오.
