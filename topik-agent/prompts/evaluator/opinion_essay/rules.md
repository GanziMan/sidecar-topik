[채점 세부 원칙: 감점제 적용]

1. 공식 가이드라인을 기준으로 하되, 발견된 오류마다 점수를 차감하는 방식으로 엄격히 채점하십시오.
2. **반드시 `reasoning` 필드에 점수 계산 과정을 상세히 서술해야 합니다.** (예: "질문 1개 누락(-4점), 문체 오류(-5점) 확인됨.")

3. 내용 및 과제 수행 (12점 만점) 감점 기준표:
   - 문제에서 제시한 질문(과제) 누락: 질문 1개당 **-4점 감점** (가장 치명적)
   - 주제와 무관한 내용 서술: **-3~5점 감점**
   - 주장이 불분명하거나 근거가 빈약함: **-2점 감점**

4. 글의 전개 구조 (12점 만점) 감점 기준표:
   - 도입-전개-마무리(서론-본론-결론) 구조 미비: **-4점 감점**
   - 문단 구분이 없거나(통글) 부적절함: **-3점 감점**
   - 문장 간 연결(접속사 등)이 부자연스러움: **-1~2점 감점**

5. 언어 사용 (24점 만점) 감점 기준표:
   - 종결 어미 오류(해요체/습니다체 사용): 발견 즉시 **-5점 감점**
   - 비격식 표현/구어체 사용: 발견 시 **-2점 감점**
   - 단순/초급 어휘의 반복 사용: **-2점 감점** (다양한 표현 권장)
   - 띄어쓰기/맞춤법 오류: 1개당 1점 감점 (단, 최대 -5점까지만 감점)
   * 예시: 질문 하나를 안 쓰고(-4점), 해요체를 썼다면(-5점) -> 50점 만점에서 총 9점 이상 감점.

6. 'strengths'와 'weaknesses'에 지적한 내용은 반드시 점수에 반영되어야 합니다. 지적해놓고 만점을 주지 마십시오.
