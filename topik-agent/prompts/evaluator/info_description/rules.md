[채점 세부 원칙: 감점제 적용]

1. 공식 가이드라인을 기준으로 하되, 발견된 오류마다 점수를 차감하는 방식으로 엄격히 채점하십시오.
2. **반드시 `reasoning` 필드에 점수 계산 과정을 상세히 서술해야 합니다.** (예: "비격식 표현 1개(-2점), 정보 누락 1개(-2점) 확인됨.")
3. 언어 사용 (16점 만점) 감점 기준표:
   - 띄어쓰기/맞춤법 오류: 1개당 1점 감점 (단, 최대 -5점까지만 감점)
   - 문법 오류(조사 등): 1개당 1점 감점
   - 종결 어미 오류(해요체/습니다체 사용): 발견 즉시 **-5점 감점** (치명적 오류)
   - 비격식 표현 사용('~것 같다', '~하고' 등 구어체): 발견 시 **-2점 감점** (부분적 문체 오류)
   * 주의: 단순히 표현이 매끄럽지 않다는 이유로 감점하지 마십시오. 명확한 규정 위반(오타, 비문, 구어체)일 때만 감점하십시오.
   * 예시 1: 띄어쓰기 3개 틀리고 '~것 같다' 사용 -> 16점 -3점 -2점 = 11점
   * 예시 2: 전체적으로 '해요체' 사용 -> 16점 -5점 = 11점
4. 내용 수행 (7점 만점) 감점 기준표:
   - 그래프/도표의 핵심 정보 누락: 1개당 **-2점 감점** (공식 기준: 과제 수행 미흡)
   - 자료에 없는 주관적 내용 추가: **-1점 감점**
5. 'strengths'와 'weaknesses'에 지적한 내용은 반드시 점수에 반영되어야 합니다. 지적해놓고 만점을 주지 마십시오.
