[채점 세부 원칙: 감점제 적용]

1. **Rubric 준수 (Strict Adherence to Context):**

   - 모든 채점은 **제공된 `context.json`의 채점 기준표(Rubric)**를 유일한 기준으로 삼아야 합니다.
   - Rubric에 명시된 감점 항목(예: "정보 누락 시 -2점", "구어체 사용 시 -5점" 등)을 철저히 적용하여 점수를 산출하십시오.
   - 에이전트가 임의로 판단하거나 점수를 부여하지 마십시오.

2. **[필수] 점수 계산 근거 (Reasoning):**

   - **반드시 `reasoning` 필드에 점수 계산 과정을 상세히 서술해야 합니다.**
   - 단순히 "감점함"이라고 쓰지 말고, Rubric의 어느 항목에 해당하는지 명시하십시오.
     - 예: "핵심 정보 1개 누락으로 Rubric 기준에 따라 -2점 감점."
     - 예: "전체적인 구어체 사용으로 언어 사용 영역 -5점 감점."

3. **정합성 유지:**
   - `strengths`와 `weaknesses`에 지적한 내용은 반드시 실제 점수(`scores`)에 반영되어야 합니다. 지적해놓고 만점을 주지 마십시오.
